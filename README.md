This is repository contains the experimental code and data used for the paper "From Semantics to Reasoning -- Modelling Pragmatic Argumentative Inferences". The repository does not contain any models, so you need to train / download the required models accordingly.

## Running the code in this repository
The code in this repository requires adjustments to be run in your specific environment. Please ensure that all paths are up to data for your specific case. In addition, the corpus preprocessing requires you to have the corresponding corpora downloaded to your device. We do not provide the raw corpora here for licensing reasons.
