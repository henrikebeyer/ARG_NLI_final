# From Semantics to Reasoning -- Modelling Pragmatic Argumentative Inferences

This repository contains the experimental code, data, and analysis files used for the paper *"From Semantics to Reasoning -- Modelling Pragmatic Argumentative Inferences"*. 

**Note on Models and Data:** This repository does not contain any trained models; you will need to train or download the required models independently. Additionally, the corpus preprocessing requires you to have the corresponding source corpora downloaded to your local device. We do not provide the raw source corpora here due to licensing restrictions.

## Repository Structure

* `ArgNLI_corpus/`: Data and files related to the ArgNLI corpus.
* `NLI_fine-tuning_results/`: Outputs and logs from the NLI fine-tuning experiments.
* `annotation/`: Files related to the feature analysis and annotation process.
* `code/`: Core scripts, including feature extraction.
* `feature_analysis/`: Scripts and data for analyzing the linguistic features.
* `inferential_embeddings_results/`: Outputs and evaluation data from the inferential embedding training.
* `quantitative_analysis/`: Scripts handling the statistical and quantitative evaluations.

## Running the Code

The code in this repository requires adjustments to be run in your specific environment. It is provided as-is for methodological transparency. 
Please ensure that all file paths are up to date for your specific local setup before executing any scripts. As mentioned above, you will also need to acquire the raw corpora independently to run the preprocessing pipelines.
